== Description ==

**Gestión de Asistencia** permite el *manejo de usuarios, clases, asistencias y pagos*. Podrás insertar alumnos y clases y después manejar las asistencias a clase y los pagos de las clases incluso desde tu móvil.

== Installation ==

1. Poner el correo en el que se recibiran los avisos de pagos realizados sección de *Opciones*.
2. Al instalar el plugin se instalan las tablas necesarias en la base de datos.
3. Para empezar a utilizar las funcionalidades de *pagos* y *asistencias* primero hay que dar de alta usuarios, clases y matricularlos a cada uno donde corresponda.
4. Para poder poner la sección de *pagos* y *asistencia* en cualquier página de tu web tienes que usar los shortcodes *[tabla_pagos_front]* y *[tabla_asistencia]* respectivamente.

== Frequently Asked Questions ==

=¿Este plugin sirve para cualquier clase?=
Sí, este plugin es un plugin neutro en el que se ponen los usuarios y las clases y a partir de esto se puede tener un seguimiento de las asistencias y los pagos de dichas clases en diferentes días y meses. Puede ser usado por cualquier tipo de clase.
